# GPA/CGPA Calculator in HTML

## Overview

This GPA/CGPA Calculator is a simple and user-friendly web application built using HTML, CSS, and JavaScript. It helps students calculate their GPA or CGPA on a 4.00 scale quickly and accurately by entering subject GPAs.

## Features

* Calculate GPA/CGPA on a 4.00 scale
* Simple and clean user interface
* Dynamic subject input support
* Instant calculation results
* Input validation for accurate calculations
* Responsive design for desktop and mobile devices

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How It Works

1. Enter the GPA for each subject.
2. Add more subjects if needed.
3. Click the "Calculate GPA/CGPA" button.
4. The calculator will display the overall GPA/CGPA out of 4.00.
